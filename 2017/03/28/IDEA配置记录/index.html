IDEA配置记录

记录IntelliJ IDEA使用过程中需要注意的配置。

创建maven项目

New Module/New Project ->

填写GroupId、ArtifactId和Version ->

接着这里建议在Properties中添加一个参数

archetypeCatalog=internal

不加这个参数，在maven生成骨架的时候将会非常慢，有时候直接卡住。来自网上的解释：

archetypeCatalog表示插件使用的archetype元数据，不加这个参数时默认为remote，local，即中央仓库archetype元数据，由于中央仓库的archetype太多了所以导致很慢，指定internal来表示仅使用内部元数据。

之后Next，完成创建。

Tomcat中文乱码问题

Tomcat控制台中文乱码，解决方案：

打开Tomcat配置对话框，选择Startup/Connection面板，勾选Pass environment variables，并添加一个environment variable：

JAVA_TOOL_OPTIONS = -Dfile.encoding=UTF-8

选择Server面板，在VM options中添加：

-Dfile.encoding=UTF-8

点击确定，重启Tomcat。

IDEA结合Tomcat运行工程，页面出现中文乱码，在确保工程代码里编码都已经设置过UTF-8，可以试下添加这个配置。

如上述全部尝试还是乱码则需要修改配置文件，进入idea的安装目录, 进入bin目录下。找到idea.exe.vmoptions 和 idea64.exe.vmoptions，在最后一行加入-Dfile.encoding=UTF-8，两个文件最好都加上。
